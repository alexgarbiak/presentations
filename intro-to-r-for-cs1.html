Probability Distributions üé∞

  • R has in-built functions for probability distributions:
    • d<distribution-name> \(:=\) density (PDF), i.e. \(f_X(x)\)
    • p<distribution-name> \(:=\) probability distribution cumulative function (CDF), i.e. \(F_X(x) =\boldsymbol{P}(X \leq x)\)
    • q<distribution-name> \(:=\) quantile function, i.e. return \(x\) such that \(\boldsymbol{P}(X \leq x) = p\)
    • r<distribution-name> \(:=\) random deviates, i.e. (psuedo) random number generator for a given distribution
    • Where <distribution-name> \(=\) Normal, uniform, lognormal, Student‚Äôs t, Poisson, binormal, Weibull ‚Ä¶ see ?distributions() for more information

Probability Distributions Examples üé∞

R Code Definition
rnorm(1) Generates \(x_1\) where \(X \sim \mathcal{N}(0,\,1)\)
rnorm(y, mean=10, sd=2) Generates \(\{y_1,\,y_2,\,\dots\}\) with \(Y \sim \mathcal{N}(10,\,2^2)\)
runif(3, min=5, max=10) Generates \(\{z_1,\,z_2,\,z_3\}\) where \(Z \sim \mathcal{U}(5,\,10)\)
dbinom(4, size=5, prob=0.5) Computes \(\boldsymbol{P}(X = 4)\) where \(X \sim \mathcal{Bin}(5,\,0.5)\)
pgamma(0.2, shape=2, rate=2)
*See footnote		 Computes \(F_Y(0.2)\) where \(Y \sim \mathcal{\Gamma}(2,\,2)\), i.e.¬†\(\boldsymbol{P}(Y\leq 0.2)\)
qexp(0.5, rate = 2) Determines smallest value of \(z\) for \(\boldsymbol{P}(Z \leq z) = 0.5\) where \(Z \sim Exp(2)\)

*This is how the \(\Gamma\) distribution is parametrised in ‚ÄúFORMULAE AND TABLES for EXAMINATIONS of THE INSTITUTE and FACULTY OF ACTUARIES‚Äù

Statistical inference

Population mean confidence intervals üî≠

Let‚Äôs calculate confidence intervals for ‚Äúthe mean ‚Ä¶ of a normal distribution‚Äù using Student‚Äôs t-Test using the t.test() function:

set.seed(42) # To fix result
# We create x_1, x_2, ... x_100 with X ~ N(0,1)
x <- rnorm(100)

# 99% CI for population mean:
as.vector(
  t.test(x, conf.level = 0.99)$conf.int
  )
## [1] -0.2409877  0.3060174

Population variance confidence intervals üî≠

We can also calculate confidence intervals for ‚Äúthe variance of a normal distribution‚Äù using the \(\chi^2\) distribution:

set.seed(42) # To fix result
# We create x_1, x_2, ... x_100 with X ~ N(0,1)
x <- rnorm(100)

degrees_freedom <- length(x) - 1
sample_var_x <- var(x)
# 95% CI for population variance:
c(
  degrees_freedom * sample_var_x / qchisq(0.975, degrees_freedom),
  degrees_freedom * sample_var_x / qchisq(0.025, degrees_freedom)
)
## [1] 0.8359784 1.4634191

Generalised linear models üåå

To ‚Äúfit a generalised linear model to a data set‚Äù in \(R\) we make use of the glm() function. Here we generalise:

  • Predicted variable \(Y\) follows a exponential family distribution
  • Explanatory variables form a linear predictor \(\beta_0 + \beta_1X_1, \, \beta_2X_2, \, \dots, \, \beta_nX_n\)
  • We link \(Y\) and the linear predictor using a link function
# NOT EVALUATED
glm_model <- 
  glm(formula = y ~ x1 + x2 + x3,
      family = Gamma(link = "inverse"), 
      data = data_set
      )

CS1 Question 1 üìî

  • Given a sample from \(X \sim \mathcal{N}(\mu,\,\sigma^{2})\),

  1. Calculate the sample mean, \(\bar{x}\)

  2. Calculate a 95% confidence interval (‚ÄúCI‚Äù) for \(\mathbb{E}[X]\)

  3. Calculate a 95% CI for \(\mathbb{E}[X]\) using a bootstrap method with a sample size of 10,000

CS1 Question 1: Answer (Part 1) üìñ

# We have X equal to:
x <- 
  c(0.20, 0.40, 1.20, 0.55, 0.75, 
    1.10, 0.50, 0.70, 0.55, 0.60, 
    0.65, 0.55, 0.50, 0.50, 0.45, 
    0.65, 0.55, 0.30, 0.50, 0.45
    )

# 1. Expected value of X is:
mean(x)
## [1] 0.5825

CS1 Question 1: Answer (Part 2) üìñ

Given \(X \sim \mathcal{N}(\mu,\,\sigma^{2})\) with unknown population variance, a 95% CI for \(\mathbb{E}[X]\) is:

confidence_level <- 0.95
confidence_interval <- 
  c(
    (1 - confidence_level)/2,
    1 - (1 - confidence_level)/2
    )
degrees_freedom <- length(x) - 1
critical_value <- qt(confidence_interval, degrees_freedom)
mu <- mean(x)
sigma <- sqrt(var(x))

c(
  mu - abs(critical_value[1]) * sigma / sqrt(length(x)),
  mu + abs(critical_value[1]) * sigma / sqrt(length(x))
)
## [1] 0.4737234 0.6912766

CS1 Question 1: Answer (Part 3) üìñ

We can also calculate a 95% CI for \(\mathbb{E}[X]\) using a bootstrap method:

set.seed(42)
sample_size <- 10000
confidence_level <- 0.95
confidence_interval <- 
  c(
    (1 - confidence_level)/2,
    1 - (1 - confidence_level)/2
  )
x_bootstrapped <- 
  replicate(sample_size,
            mean(sample(x, replace = TRUE))
            )
quantile(x_bootstrapped, probs = confidence_interval)
##  2.5% 97.5% 
##  0.49  0.69
